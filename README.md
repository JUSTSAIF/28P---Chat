###### ============================ Configure ============================

### first run : run.py 

##### ========== Commands ==========

<br>

```
[--c] --conf
[--r] --reconfig
[--p] --peerhost
```

### run.py  --conf
##### `this will show your data like : name,password hash, your host,peer host.....`

### run.py  --reconfig
##### `this will reset your data`

### run.py  --peerhost  "set your Friend tor host here"
##### `This will update the host you are talking to`
